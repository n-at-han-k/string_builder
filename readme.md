# string_builder

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/n-at-han-k/string_builder)

Ruby method chains to any string format. 113 lines. Zero dependencies.

## Usage

Please see the [project documentation](https://n-at-han-k.github.io/string_builder/) for more details.

  - [Getting Started](https://n-at-han-k.github.io/string_builder/guides/getting-started/index) - This guide walks you through installing string_builder and building your first DSL.

  - [CLI Builders](https://n-at-han-k.github.io/string_builder/guides/cli-builders/index) - This guide covers building command-line tool DSLs (git, docker, terraform, aws).

  - [Query Builders](https://n-at-han-k.github.io/string_builder/guides/query-builders/index) - This guide covers building query and data-access DSLs (SQL, JSONPath, .env files).

  - [Web Builders](https://n-at-han-k.github.io/string_builder/guides/web-builders/index) - This guide covers building web-related DSLs (HTML, CSS selectors, URLs, Makefile targets).

  - [Multi-Render](https://n-at-han-k.github.io/string_builder/guides/multi-render/index) - This guide covers rendering the same method chain in different formats by swapping handlers.
